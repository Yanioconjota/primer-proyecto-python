# Primer Proyecto Python

Repositorio de aprendizaje de Python desde cero. Contiene scripts de práctica y una guía de referencia unificada, todo escrito en español.

## Estructura

```
primer-proyecto-python/
├── guides/
│   ├── python-guia-completa.md   # Guía unificada (17 temas + Q&A)
│   └── index.md                  # Índice con links a cada sección
│
├── primer_script.py              # Primeros pasos con print() e input()
├── variables.py                  # Declaración y uso de variables
├── integers_floats.py            # Números enteros y decimales
├── redondeo.py                   # Redondeo de números
├── conversiones.py               # Conversión entre tipos de datos
├── formatear_cadenas.py          # f-strings y .format()
├── propiedades_string.py         # Propiedades e indexado de strings
├── metodos_string.py             # Métodos de string
├── fragmentar.py                 # Slicing de strings y listas
├── listas.py                     # Listas
├── diccionarios.py               # Diccionarios
├── tuples.py                     # Tuples
├── sets.py                       # Sets
├── booleanos.py                  # Booleanos
├── operadores.py                 # Operadores aritméticos y lógicos
├── operadores_comparacion.py     # Operadores de comparación
├── operadores_logicos.py         # Operadores lógicos
├── control_flujo.py              # if / elif / else
├── match.py                      # match / case (Python 3.10+)
├── loop_for.py                   # Loops for
├── indice.py                     # Ejercicio: índice de palabras
├── analizador_texto.py           # Ejercicio: análisis de texto
├── calculo_comisiones.py         # Ejercicio: cálculo de comisiones
└── cervezeria.py                 # Ejercicio: menú de cervezas
```
