# Primer Proyecto Python

Repositorio de aprendizaje de Python desde cero. Contiene scripts de práctica y guías de referencia escritas en español.

## Estructura

```
primer-proyecto-python/
├── guides/              # Guías de referencia por tema
├── primerScript.py      # Primeros pasos con print() e input()
├── variables.py         # Declaración y uso de variables
├── integers-floats.py   # Números enteros y decimales
├── operadores.py        # Operadores aritméticos y lógicos
├── formatear-cadenas.py # f-strings y .format()
├── conversiones.py      # Conversión entre tipos de datos
├── redondeo.py          # Redondeo de números
└── calculo-comisiones.py / cervezeria.py   # Ejercicios prácticos
```
